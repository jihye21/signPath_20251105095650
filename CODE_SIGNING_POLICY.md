# Code Signing Policy

This project uses **free code signing provided by [SignPath.io](https://signpath.io)**,  
with certificates issued by **SignPath Foundation**.

## Team Roles
- **Committers and Reviewers:** [GitHub Members](https://github.com/jihye21)
- **Approvers:** [Repository Owners](https://github.com/jihye21?tab=repositories)

## Privacy Policy
This program will not transfer any user information to external systems  
unless specifically requested by the user during operation.

## Build Integrity
All signed artifacts are built from verified source code and uploaded to SignPath.io  
for signing through automated pipelines. No proprietary binaries are included.
